# LSSP_PCP1 - Industrial Operation Data Analysis

Análise de dados de operações industriais usando Jupyter Notebook.

## Estrutura
- `industrial operation data analisys LSSP_PCP1.ipynb` - Notebook principal
- `plots/` - Gráficos gerados

## Como usar
1. Criar venv: `python -m venv venv`
2. Ativar: `venv\Scripts\activate`
3. Instalar dependências: `pip install -r requirements.txt`
4. Abrir notebook: `jupyter notebook`
